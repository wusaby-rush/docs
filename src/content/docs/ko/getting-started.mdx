<h2>Astro란 무엇인가요?</h2>

Astro는 **콘텐츠 중심의** **빠른** 웹 사이트를 제작하기 위한 **올인원(all-in-one)** **웹 프레임워크**입니다. 

## 주요 기능

- **컴포넌트 아일랜드:** 더 빠른 웹사이트 구축을 위한 새로운 웹 아키텍처 입니다.
- **서버 우선 API 설계:** 사용자 디바이스에서 값비싼 수화 작업 제거합니다.
- **Zero JS 지원:** 속도를 저하시키는 자바스크립트 런타임 오버헤드가 없습니다.
- **엣지 지원:** Deno 또는 Cloudflare와 같은 글로벌 엣지 런타임을 지원하며, 어디서나 배포할 수 있습니다.
- **사용자 정의 기능 제공:** Tailwind, MDX 및 100여개 이상의 통합 기능을 선택할 수 있습니다.
- **다양한 UI 프레임워크 사용가능:** React, Preact, Svelte, Vue, Solid, Lit 등을 지원합니다.

[Why Astro](/ko/concepts/why-astro/)글을 통해 Astro의 특징을 자세히 살펴보세요. ✨

## 첫 번째 프로젝트를 생성하기

`create astro` CLI 생성 마법사를 사용하여 새로운 Astro 프로젝트를 로컬에서 시작하고 실행하세요!

<PackageManagerTabs>
  <Fragment slot="npm">
  ```shell
  # npm으로 새 프로젝트 만들기
  npm create astro@latest
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```shell
  # pnpm으로 새 프로젝트 만들기
  pnpm create astro@latest
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```shell
  # yarn으로 새 프로젝트 만들기
  yarn create astro
  ```
  </Fragment>
</PackageManagerTabs>

[Astro를 CLI로 자동 설치하기](/ko/install/auto/)에서 CLI 도구를 사용하여 Astro를 설치하고, 기존 Astro GitHub 저장소에서 새 프로젝트를 생성하고, 수동으로 Astro를 설치하는 방법에 대한 전체 단계별 지침이 나와 있습니다.
