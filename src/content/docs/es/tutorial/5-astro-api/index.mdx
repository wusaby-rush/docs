## ¿Adónde vamos?

En esta unidad, mejorarás tu blog con una página de índice, páginas de etiquetas y un feed de RSS.

Por el camino, aprenderás a utilizar:
- `Astro.glob()` para acceder a los datos de los archivos de tu proyecto
- `getStaticPaths()` para crear varias páginas (rutas) a la vez
- El paquete Astro RSS para crear un feed RSS.
