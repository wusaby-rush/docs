我们欢迎任何大小的贡献和任何技能水平的贡献者。作为一个开源项目，我们相信应该回馈给我们的贡献者。我们很乐意提供指导，帮助你完成 PR，进行技术写作，以及将任何想法变成现实。

## 贡献的方式

这里有很多方式可以为 Astro 文档网站做出贡献！

Astro 文档网站是 ... 一个 Astro 网站！维护它不仅需要书面内容，还需要维护 Astro 代码并解决 a11y、CSS、UI 和 UX 的问题。我们还将我们的文档提供给多种语言，因此我们需要帮助翻译整个网站。

你也可以通过在 PR 上留下评论、在 GitHub Issues 和 Discussions 中添加想法，以及参与我们的“置顶” issue 维护任务来参与其中！

每个 PR，尤其是翻译 PR，都需要审稿人！审查 PR 并留下评论、建议或批准的“LGTM！”（“Looks Good To Me！”）是加入 Team Docs 并了解更多关于 Astro 的好方法。

## 贡献者指南

我们有几个指南，其中包含有关如何开始为 Astro 文档网站做出贡献的信息！

- [主要的贡献指南](https://github.com/withastro/docs/blob/main/CONTRIBUTING.md) - 请先查阅此文档！
- [写作指南](https://github.com/withastro/docs/blob/main/WRITING.md) - 以 Astro 文档风格写作的提示。
- [翻译指南](https://github.com/withastro/docs/blob/main/TRANSLATING.md) - 有关如何帮助将 Astro 文档翻译成您的语言的信息。
- [方案](https://github.com/withastro/docs/blob/main/RECIPES.md) - 学习如何编写和贡献一个方案。

> 想要为 Astro 代码库做出贡献吗？请查看 [Astro 的贡献指南](https://github.com/withastro/astro/blob/main/CONTRIBUTING.md)。
