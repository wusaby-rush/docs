フル機能のWebアプリホスティングプラットフォーム[Google Cloud](https://cloud.google.com/)で、Astroサイトをデプロイできます。

## デプロイ方法

プロジェクトをGoogle Cloudにデプロイするには、いくつかのUIクリックが必要です（これらのアクションのほとんどは、[gcloud CLI](https://cloud.google.com/sdk/gcloud/)を使用しても実行可能です）。

### Cloud Run

1. 新しいGCPプロジェクトを作成するか、既存のプロジェクトを選択します。

2. Cloud Run APIが有効になっていることを確認します。

3. 新しいサービスを作成します。

4. Docker Hub上のコンテナを使用するか、[Cloud Build](https://cloud.google.com/build)を使用して独自のコンテナをビルドします。

5. ファイルが提供されるポートを設定します。

6. `allUsers`に`Cloud Run Invoker`という権限を新たに追加することで、パブリックアクセスを有効にします。

### Cloud Storage

1. 新しいGCPプロジェクトを作成するか、既存のプロジェクトを選択します。

2. [Cloud Storage](https://cloud.google.com/storage)に新しいバケットを作成します。

3. バケットに名前を付け、他の必要な設定を行います。

4. `dist`フォルダをバケットにアップロードするか、[Cloud Build](https://cloud.google.com/build)を使用してアップロードします。

5. `allUsers`に`Storage Object Viewer`という権限を新たに追加することで、パブリックアクセスを有効にします。

6. ウェブサイトの設定を編集し、エントリポイントとして`ìndex.html`を、エラーページとして`404.html`を追加します。
